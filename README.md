# file-organizer
a simple CLI file organizer 
you must write the correct path, for example if i want the same path that ia m in my terminal i will type ./ , and if i want a specific path i will write it full for example ~/Desktop or ~/your/path/to/your/dir/here and press enter it will generate a summary of where the files where moved and how many files and the names of the files which were moved

enjoy!
